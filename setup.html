Setup your accounts and tools

This page makes sure your environment is ready before building. Think of this as setting up a new kitchen: once your tools are in place, cooking becomes much easier.

Create your accounts

1) GitHub

Why: stores your project code and connects directly to Vercel.

  1. Go to github.com
  2. Sign up and verify your email
  3. Pick a professional username (it appears in your URLs)

2) Vercel

Why: hosts your Next.js app online for free.

  1. Go to vercel.com
  2. Sign up using GitHub
  3. Authorize GitHub access and stay on the free Hobby plan

3) InstantDB

Why: provides backend data and Magic Code login so each user has their own account/data.

  1. Go to instantdb.com
  2. Create account and verify email

Create your InstantDB app and copy App ID

  1. Open instantdb.com/dash
  2. Click Create App
  3. Name it something like ecommerce-negotiation-platform
  4. Copy the App ID (a long ID string) and save it in notes

Mental model: the App ID is like your database address label. Your app uses it to send requests to the correct InstantDB project. It is public, not a secret password. 

# Example shape (your value will be different)
566ffee5-e773-4b68-a132-29678f8f6eea

Install Node.js

Node.js lets your computer run JavaScript during development. Without it, you cannot run `npm` commands or start your local app preview server.

  1. Visit nodejs.org
  2. Install the LTS version
  3. Open terminal and run:
node --version
npm --version

If you see version numbers, setup is correct.

Install and configure Cursor

  1. Download from cursor.com
  2. Install and sign in
  3. Open Settings -> Features -> Docs -> Add Doc
  4. Add: https://www.instantdb.com/docs

Why this matters: adding docs gives AI fresh, tool-specific instructions and usually produces better InstantDB code.

Next step

Continue to Build to generate your app with a customized prompt.

Prerequisites and Setup

This page gets your foundation ready. Think of this as setting up your kitchen and tools before cooking. Once done, you can build confidently with far fewer errors.

1) Create Your Accounts

GitHub

GitHub stores your code and version history. Vercel connects to it for auto deployment.

  1. Go to github.com.
  2. Create an account and verify your email.
  3. Choose a professional username (it appears in URLs).

Vercel

Vercel hosts your app on the internet so anyone can open it by URL.

  1. Go to vercel.com.
  2. Sign up using GitHub.
  3. Use the free Hobby plan.

InstantDB

InstantDB is your backend service for data and authentication.

  1. Go to instantdb.com.
  2. Create an account and verify your email.

2) Create Your InstantDB App

  1. Open instantdb.com/dash.
  2. Click Create App.
  3. Name it something clear, for example negotiation-platform.
  4. Copy your App ID and store it in your notes.

Why App ID matters: It is like your database's mailing address. Your app uses it to connect to the correct data space.

Example App ID:
566ffee5-e773-4b68-a132-29678f8f6eea

The App ID is public and safe in client code. Real protection comes from InstantDB permissions rules.

3) Install Node.js

Node.js lets your computer run JavaScript tools locally. During development, this is what powers commands like npm run dev.

  1. Go to nodejs.org.
  2. Download the LTS version.
  3. Install it with default settings.
  4. Verify in terminal:
node --version
npm --version

A terminal is a text control panel for your computer. It may look unfamiliar now, but you only need a handful of commands, and this tutorial tells you exactly what to run.

4) Install Cursor

Cursor is where you direct AI to generate and fix code. You do not need to type large amounts of code yourself to succeed in this project.

  1. Download from cursor.com.
  2. Install and open it.
  3. Complete sign-in and initial setup.

5) Configure Cursor Docs for Better AI Answers

AI output quality improves when your editor can reference official docs. This matters most for newer tools like InstantDB.

  1. Open Cursor Settings (Cmd+, on Mac or Ctrl+, on Windows).
  2. Go to Features -> Docs.
  3. Click Add Doc.
  4. Add https://www.instantdb.com/docs.
  5. Optionally add https://nextjs.org/docs and https://tailwindcss.com/docs.

Checkpoint

You are ready to build when you have: all three accounts, Node installed, Cursor installed, and your App ID copied.
