
# E-Commerce Negotiation Platform

## One-Sentence Summary
This app offers a platform for a customer to negotiate with retailer company's computer agent when purchase online

## The Problem solved
The app can help a company to sell products with a flaxible price corresponding to customer's personal requirement

## User Types
E-Commerce customers

## Core Features

- Administrator can setup what product to negotiate
- Set up user and agent's personal preference about products
- Exchange offers for product price or other features
- Accept offers if user liked it or satisfied pre-defined agent's preference

## What Users See

### Screen1: userDemographic
Ask user demographic information and save
### Screen2: product
a popup window ask administrator what product to negotiate and what features can be under negotiation (such as laptop, features are CPU, hard drive, RAM, screen size)
### Screen3: preference
Set up weight distribution for all the features. All the feature's weight should add up to 100%
### Screen4: featureOptions
Set up weight distribution for the options of each feature
### Screen5: negotiation
The user and agent exchange offers around the features.

## Example Scenario

A user first click start button to start a negotiation. The demographic page ask for user's name, age, email, gender, etc.. Then the screen3 popup ask for user's preference. Screen4 ask for user's options weight. Then the user can start a negotiation with computer agent. The computer agent can use current AI agent's API.
