Deploy your app to Vercel

Deployment means putting your app on an always-on server so others can use it from a URL.

Push your code to GitHub

  1. Create a new public repository at GitHub
  2. Open project in Cursor
  3. Ask Cursor to help push your code with your repo URL

Deploy with Vercel

  1. Sign in to Vercel
  2. Add New Project and select your repository
  3. Add environment variable:
Name: NEXT_PUBLIC_INSTANT_APP_ID
Value: your-real-instantdb-app-id
  1. Click Deploy
  2. Open the live URL after build succeeds

Why env vars? They keep configuration outside code and easy to change between local and production.

Post-deploy checklist

  • Magic Code login works
  • Admin can create template data
  • User can start session and exchange offers
  • Saved data remains after refresh
