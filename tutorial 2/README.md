# E-Commerce Negotiation Platform - Build Tutorial

This is your personalized tutorial for building the **E-Commerce Negotiation Platform** using Next.js, InstantDB, Tailwind CSS, and Vercel.

## Viewing This Tutorial

### Quick Start: View Locally
1. Open `index.html` in your web browser
2. Navigate through the pages using the top menu

### Host on GitHub Pages (Recommended)

#### Step 1: Create a GitHub Repository
1. Go to [github.com](https://github.com) and sign in
2. Click the **+** button -&gt; **New repository**
3. Name it (example: `ecommerce-negotiation-tutorial`)
4. Select **Public**
5. Click **Create repository**

#### Step 2: Upload Files
1. On your repository page, click **uploading an existing file**
2. Drag all files from this folder:
   - `index.html`
   - `setup.html`
   - `build.html`
   - `deploy.html`
   - `reference.html`
   - `styles.css`
   - `scripts.js`
   - `README.md`
3. Click **Commit changes**

#### Step 3: Enable GitHub Pages
1. Go to **Settings** -&gt; **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Select **main** branch and **/ (root)** folder
4. Click **Save**

#### Step 4: View Your Tutorial
After a few minutes, your tutorial will be live at:

```text
https://YOUR-USERNAME.github.io/REPOSITORY-NAME/
```

---

## Next Steps
1. Follow the tutorial from `index.html` to `reference.html`
2. Copy the Build Prompt from `build.html`
3. Build and deploy your actual app to Vercel
4. Share your finished app
# E-Commerce Negotiation Platform - Build Tutorial

This is your personalized tutorial website for building your E-Commerce Negotiation Platform using Next.js, InstantDB, Tailwind CSS, and Vercel.

## Viewing This Tutorial

### Quick Start: View Locally
1. Open `index.html` in your web browser.
2. Use the top navigation bar to move through all 5 pages.

### Host on GitHub Pages (Recommended)

#### Step 1: Create a GitHub Repository
1. Go to [github.com](https://github.com) and sign in.
2. Click the **+** button then **New repository**.
3. Name it (example: `negotiation-platform-tutorial`).
4. Select **Public**.
5. Click **Create repository**.

#### Step 2: Upload Files
1. On your repository page, click **uploading an existing file**.
2. Upload all files from this folder:
   - `index.html`
   - `setup.html`
   - `build.html`
   - `deploy.html`
   - `reference.html`
   - `styles.css`
   - `scripts.js`
   - `README.md`
3. Click **Commit changes**.

#### Step 3: Enable GitHub Pages
1. Go to **Settings** then **Pages**.
2. Under **Source**, choose **Deploy from a branch**.
3. Choose **main** branch and **/ (root)** folder.
4. Click **Save**.

#### Step 4: View Your Live Tutorial
After a few minutes, your tutorial will be available at:

`https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

## Next Steps

1. Follow the tutorial pages in order.
2. Copy the personalized Build Prompt from `build.html`.
3. Build your app in Cursor.
4. Deploy your app to Vercel and share it.

## Notes

- This tutorial site is static HTML/CSS/JS and works well with GitHub Pages.
- The app you build from the tutorial uses Next.js App Router, InstantDB auth/data, and Vercel deployment.
