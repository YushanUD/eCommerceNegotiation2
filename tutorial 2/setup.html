Setup your accounts and tools

Think of this page as preparing your kitchen before cooking. Once setup is done, building is much smoother.

Create your accounts

GitHub

Why: stores your code and connects to Vercel deployment.

  1. Go to github.com
  2. Sign up and verify email
  3. Pick a professional username

Vercel

Why: hosts your app online for free.

  1. Go to vercel.com
  2. Sign up with GitHub
  3. Use Hobby plan

InstantDB

Why: backend data + Magic Code authentication.

  1. Go to instantdb.com
  2. Create account and verify email

Create your InstantDB app and App ID

  1. Open instantdb.com/dash
  2. Click Create App
  3. Copy your App ID and save it

Mental model: App ID is your database address label. It is public and safe to expose.

# Example format only
566ffee5-e773-4b68-a132-29678f8f6eea

Install Node.js

Node.js allows your computer to run JavaScript tooling like npm and local dev servers.

  1. Go to nodejs.org
  2. Install LTS version
  3. Verify:
node --version
npm --version

Install and configure Cursor docs

  1. Install Cursor from cursor.com
  2. Open Settings -> Features -> Docs -> Add Doc
  3. Add https://www.instantdb.com/docs

Why: giving Cursor current docs improves generated code accuracy.
